Hello World

This is the first post about Hexo blog. There are some steps about how to create a bolg using GitHub Page.

Quick Start

Install git, nodejs, npm, hexo

$ sudo apt-get install git
$ sudo apt-get install npm
$ sudo apt-get install nodejs

If the nodejs version is below 18.x, please update it.

$ curl -sL https://deb.nodesource.com/setup_18.x | sudo -E bash -
$ sudo apt-get install -y nodejs

Install hexo

$ npm install hexo -g

Test hexo version

$ hexo -v

Install hexo git deployer tools

$ npm install --save hexo-deployer-git

Github config

Generate ssh key

$ ssh-keygen -t rsa -C "mailaddress@xx.com"

Create a new ssh-key, then put the key(in the /home/xx/.ssh/id_rsa.pub) into the ssh-rsa value.

Test it

$ ssh -T git@github.com

Config the github account

$ git config --global user.name "username" 
$ git config --global user.email "username@xx.com"

Create a new post

Start the hello world page

$ hexo init  #init the blog

$ hexo g     #generate the static pages

$ hexo s     #start the local site http://localhost:4000/

Update the config, please edit the _config.yml.

Upload to Github

Create a new respository, name is username.github.io.

Config file(_config.yml), update deploy section as follow.

deploy:
  type: git
  repository: git@github.com:username/username.github.io.git
  branch: main

upload to Github

$ hexo clean
$ hexo g
$ hexo d

Some hexo command

$ hexo new "postName"
$ hexo new page "pageName"
$ hexo generate
$ hexo server
$ hexo deploy
$ hexo help
$ hexo version
