Git Tips

Git for first upload

$ echo "# Test" >> README.md
$ git init
$ git add .
$ git add README.md
$ git commit -m "commit info"
$ git remote add main https://github.com/xxxxxxx/Test.git
$ git push -u main master

Git for update

$ git status
$ git add .
$ git commit -m "commit info"
$ git pull
$ git push -u main master
