2-Mount Disk

2.1-Find all partition

sudo fdisk -lu

This command will find the new partition info.

2.2-Partition the disk

sudo fdisk /dev/sdb
m
n
p
w

2.3-Find the new partition

sudo fdisk -lu

2.4-Format the disk

sudo mkfs -t ext4 /dev/sdb6

2.5-Mount

cd /home
sudo mkdir /home/newdir
sudo mount /dev/sdb6 /home/newdir
df -h

2.6-Setting auto mount

sudo gedit /etc/fstab
add the new line:
/dev/sdb6   /home/newdir    ext4    defaults    0   1
