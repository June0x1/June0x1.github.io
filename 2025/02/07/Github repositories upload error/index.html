error Permission denied (publickey). fatal Could not read from remote repository. Please

Reason

SSH key failure or error

How to Solve

Generate a new SSH key

ssh-keygen -t rsa -C "xxx@xxx.com"

Find the id_rsa.pub file(the path is \home.ssh), and paste it’s contents into the GitHub(settings\ssh and gpg keys) content.

And then push again.
